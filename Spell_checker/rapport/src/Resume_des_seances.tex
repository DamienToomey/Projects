\subsection{Choix des conventions de nommage dans les .c et .h}

Une fois la conception détaillée terminée, nous avons dû choisir des
conventions de nommage pour pouvoir développer notre partie du code
chacun de notre côté tout en restant cohérent avec le code des
autres.
Nous avons donc choisie les conventions suivantes :

\begin{description}
\item[TAD Mot : ] Nous adopterons les conventions suivantes :
  \begin{itemize}
  \item[.] Le type Mot sera nommé : \emph{M\_Mot}.
  \item[.] Les fonctions et procédures du TAD
    Mot seront nommées de la façon suivante : \\
    \emph{M\_nomDeLaFonction ou M\_nomDeLaProcédure}.
  \end{itemize}
\item[TAD Dictionnaire : ] Nous adopterons les conventions suivantes :
  \begin{itemize}
  \item[.] Le type Dictionnaire sera nommé :
    \emph{D\_Dictionnaire}.
  \item[.] Les fonctions et procédures du TAD
    Dictionnaire seront nommées de la façon
    suivante : \\
    \emph{D\_nomDeLaFonction ou D\_nomDeLaProcédure}.
  \end{itemize}
\item[TAD CorrecteurOrthographique : ] Nous adopterons les
  conventions suivantes :
  \begin{itemize}
  \item[.] Le type CorrecteurOrthographique sera
    nommé :
    \emph{CO\_CorrecteurOrthographique}.
  \item[.] Les fonctions et procédures du TAD
    CorrecteurOrthographique seront nommées de
    la façon suivante : \\
    \emph{CO\_nomDeLaFonction ou CO\_nomDeLaProcédure}.
  \end{itemize}
\item[TAD Ensemble : ] Nous adopterons les conventions suivantes :
  \begin{itemize}
  \item[.] Le type Ensemble sera nommé :
    \emph{E\_Ensemble}.
  \item[.] Les fonctions et procédures du TAD
    Ensemble seront nommées de la façon suivante
    : \\
    \emph{E\_nomDeLaFonction ou E\_nomDeLaProcédure}.
  \end{itemize}
\item[TAD ListeChaineeMot : ] Nous adopterons les conventions
  suivantes :
  \begin{itemize}
  \item[.] Le type ListeChaineeMot sera nommé :
    \emph{LCM\_ListeChaineeMot}.
  \item[.] Les fonctions et procédures du TAD
    ListeChaineeMot seront nommées de la façon
    suivante : \\
    \emph{LCM\_nomDeLaFonction ou LCM\_nomDeLaProcédure}.
  \end{itemize}
\item[TAD ArbreN : ] Nous adopterons les conventions suivantes :
  \begin{itemize}
  \item[.] Le type ArbreN sera nommé : \emph{AbN\_ArbreN}.
  \item[.] Les fonctions et procédures du TAD
    ArbreN seront nommées de la façon suivante :
    \\
    \emph{AbN\_nomDeLaFonction ou Abn\_nomDeLaProcédure}.
  \end{itemize}

\end{description}
