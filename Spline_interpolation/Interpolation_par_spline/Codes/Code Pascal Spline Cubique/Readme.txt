Programme FonctionSplineCubique

A l'exécution du programme, un répertoire nommé "Résultats" est créé s'il n'existe pas encore, dans le répertoire où se trouve le programme.

L'utilisateur entre le nombre de points qu'il souhaite entrer (un nombre entier) puis appuie sur "Entrée".
Ensuite l'utilisateur entre l'abscise du premier point et appuie sur "Entrée" puis entre l'ordonnée de ce point et appuie sur "Entrée". L'utilisateur fait de même pour les autres points.

Une fois avoir entré tous les points, l'utilisateur peut enregistrer, les splines calculées par l'ordinateur, dans le répertoire "Résultats".
